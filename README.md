## About <a name = "about"></a>

This is an Apollo Server basic example

### Prerequisites

This projects needs the following:

    - NodeJS
    - Npm
    - Git

### Installing

To install the dependencies, please run:

```
npm install
```

## Usage <a name = "usage"></a>

To use and run queries, run the following command:

```
npm run dev
```
